## Overview

Bread provides comprehensive cryptocurrency wallet management capabilities. Each wallet supports both EVM (Ethereum Virtual Machine) and SVM (Solana Virtual Machine) networks, giving you access to a wide range of blockchain ecosystems.

### Key Features

- **Multi-chain Support**: Every wallet includes both EVM and SVM addresses
- **Fully Automated**: Configure automatic offramp, swap, and sweep functionality
- **Secure Management**: Enterprise-grade security for wallet creation and management
- **100% Gasless**: Transactions accross supported chains without worry about gas fees

### Multi-Chain Architecture

Every Bread wallet includes addresses for all major blockchains:

- **EVM Address**: Compatible with Base, and any other EVM chains
- **SVM Address**: Compatible with Solana, and any other SVM chains

This dual-address system allows your users to receive and manage assets across the most popular blockchains without needing separate wallets.

## Best Practices

<AccordionGroup>
  <Accordion title="Reference Management">
    Use meaningful reference identifiers that map to your internal user or account system. This makes it easier to track and manage wallets across your application.
  </Accordion>
  
  <Accordion title="Security Considerations">
    - Store wallet IDs securely in your database
    - Never expose wallet passcodes and API keys
    - Implement proper access controls for wallet operations
    - Use HTTPS for all API communications
    - Validate wallet ownership before performing operations
  </Accordion>
</AccordionGroup>
